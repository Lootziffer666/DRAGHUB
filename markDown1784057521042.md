Du könntest direkt korrigieren



Nicht erneut einen ganzen Satz schreiben. Du klickst auf einen Eintrag:

Der Prompt beziehungsweise der maschinenlesbare Kontext wird automatisch ergänzt:



{

  "presentation": "overlay",

  "preserveParentState": true,

  "allowNewRoute": false

}



Die Kontextliste wäre dadurch nicht nur eine Anzeige, sondern ein interaktiver Präzisionsfilter.



Noch besser: Wahrscheinliche Interpretation gegen Alternativen



Zum Beispiel:



„Die URL soll sich ändern lassen.“



Wahrscheinlich gemeint

● URL darf manuell überschrieben werden



Mögliche Alternativen

○ URL folgt immer dem Repo

○ Änderung der URL ändert auch den Projektnamen

○ Änderung gilt nur für diesen Lauf



Du klickst die richtige Interpretation an, bevor der Agent irgendetwas baut.
